# Spam-mail-prediction-Using-Machine-Learning
## Spam Mail Prediction

## Overview
This project uses Machine Learning to classify emails as Spam or Ham.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy

## Dataset
The dataset contains labeled email messages for spam detection.

## How to Run
1. Clone the repository.
2. Install required libraries:
   pip install -r requirements.txt
3. Run:
   python app.py

## Results
The model achieved 97% accuracy on the test dataset.
